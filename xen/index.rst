XEN 虚拟化
==============================


xen的文件系统是用分区呢？还是用的是文件？
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
问 :
    听说如果是用文件的话，xen的文件系统会差很多。。。。

答 : 
    2002年前用分区性能好这句话是对的；

    2002年后科技发展，硬件、磁盘缓存、内核、文件系统等发展很快，用文件可以借助 host 上文件系统和内存缓存，可以提高性能。

    测试结果两个差不多，用文件稍微强一点，但是用文件会方便很多，方便客户转移机房，转移 host
